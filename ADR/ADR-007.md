ADR-007

Frontend Folder Structure

Objetivo

Padronizar toda a estrutura React do Companion.

Decisão

src/

assets/
    css/
    icons/
    images/
    fonts/

components/
    common/
    layout/
    widgets/

pages/

services/

state/

hooks/

types/

utils/

Racional

...

Consequências

Todos os módulos deverão utilizar essa estrutura.

Nenhum módulo poderá criar novas pastas sem alteração desta ADR.

# ADR-006 - AI Agent Responsibilities

## Status

Accepted

---

## Contexto

O desenvolvimento do K.O.R.E. utiliza múltiplos agentes de IA especializados.

Cada agente possui responsabilidades bem definidas para evitar sobreposição de funções e manter consistência durante o desenvolvimento.

---

## Decisão

O projeto adota três agentes oficiais.

### Gaspar

Responsável por:

- Arquitetura.
- Specification Driven Development.
- ADRs.
- Revisões técnicas.
- Decisões arquiteturais.
- Consistência do projeto.

### Belthazar

Implementação oficial do projeto.

Ferramentas:

- Aider
- qwen2.5-coder

Responsável por:

- Implementação.
- Refatoração.
- Testes.
- Correção de bugs.

Belthazar nunca deverá alterar arquitetura ou criar requisitos.

### Melchior

Especialista em interface.

Ferramentas:

- Gemini CLI
- UI/UX Pro Max

Responsável por:

- Interface.
- Experiência do usuário.
- Componentização.
- Design System.

Melchior nunca deverá alterar regras de negócio ou arquitetura.

---

## Consequências

- Responsabilidades claramente definidas.
- Menor conflito entre agentes.
- Desenvolvimento consistente.
- Arquitetura preservada durante toda a evolução do projeto.

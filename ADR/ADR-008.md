# ADR-008 — Navegação

## Decisão

O Companion utilizará React Router com HashRouter.

## Rotas oficiais

/
#/spotify
#/google
#/weather
#/settings

## Motivação

O Companion é uma aplicação desktop baseada em Tauri.
HashRouter elimina dependência de servidor HTTP e garante funcionamento consistente em ambiente empacotado.

## Consequências

- Todas as novas telas deverão ser registradas no React Router.
- Sidebar e demais mecanismos de navegação utilizarão exclusivamente React Router.
- Não utilizar navegação baseada em estado local.
